# PDF Form Filler — User Stories

瀏覽器端 PDF 填表工具的實作任務拆解，依據 `docs/spec.md` 的 10 個實作步驟產出。

## US 一覽表

| 編號 | 標題 | 優先級 | 依賴 | 說明 |
|------|------|--------|------|------|
| US-001 | 資料庫與型別定義 | P0 | - | 共用型別 + Dexie schema |
| US-002 | 封裝 PDF 操作工具函式 | P0 | US-001 | pdf-lib 封裝（detect/fill） |
| US-003 | 實作素材 CRUD 儲存層 | P0 | US-001 | materialStore CRUD |
| US-004 | 實作儀表板與 PDF 上傳 | P0 | US-001 | 首頁、上傳、列表（含空狀態、錯誤處理） |
| US-005a | 實作 PDF 預覽元件 | P0 | US-001 | PDFViewer + 填寫頁框架 |
| US-005b | 實作表單欄位面板 | P0 | US-002, US-005a | 欄位偵測與四種輸入渲染 |
| US-006a | 實作素材庫套用面板 | P1 | US-003, US-005b | 素材側邊欄與自動填入 |
| US-006b | 實作簽名繪製與儲存工具 | P1 | US-003, US-005b | Canvas 簽名繪製與素材庫整合 |
| US-007 | 實作 PDF 匯出功能 | P1 | US-002, US-005a, US-005b | 匯出下載 |
| US-008 | 實作素材管理頁面 | P1 | US-003 | 素材 CRUD 頁面（含空狀態） |
| US-009 | 製作預設範本 PDF | P2 | US-004 | 內建範例 PDF |

US-001 (型別+DB)

  │           ├──▶ US-006a (素材套用)
  │           ├──▶ US-006b (簽名工具)
  │           └──▶ US-007 (匯出)
  ├──▶ US-003 (materialStore)
  │     ├──▶ US-006a (素材套用)
  │     ├──▶ US-006b (簽名工具)
  │     └──▶ US-008 (素材管理頁)
  ├──▶ US-004 (儀表板) ──▶ US-009 (範本)
  └──▶ US-005a (PDF預覽)
        └──▶ US-005b (欄位面板)

## 依賴說明

- **US-001** 是基礎設施，所有其他 US 直接或間接依賴它
- **US-002** 依賴 US-001 的 `PDFField` 型別
- **US-003** 依賴 US-001 的 `db` 實例與型別
- **US-004** 依賴 US-001 的 `db` 實例存取 PDF 資料
- **US-005a** 依賴 US-001 讀取 PDF，建立填寫頁框架
- **US-005b** 依賴 US-002（偵測欄位）與 US-005a（整合目標）
- **US-006a** 依賴 US-003（素材 CRUD）與 US-005b（填入目標欄位）
- **US-006b** 依賴 US-003（儲存簽名）與 US-005b（簽名嵌入目標）
- **US-007** 依賴 US-002（匯出工具）與 US-005a/US-005b（取得填寫值）
- **US-008** 依賴 US-003（materialStore API）
- **US-009** 依賴 US-004（儀表板 PresetTemplates 已實作）

## 建議開發順序

### Wave 1（基礎設施，可平行）
優先啟動 **US-001**，完成後以下三者可平行進行：
- US-002（pdfUtils）
- US-003（materialStore）
- US-004（儀表板）

### Wave 2（核心填寫流程）
- **US-005a**（PDF 預覽）← 需 US-001
- **US-005b**（欄位面板）← 需 US-002 + US-005a
- **US-008**（素材管理頁）← 需 US-003，可與 US-005 平行

### Wave 3（進階功能，可平行）
- **US-006a**（素材套用）← 需 US-003 + US-005b
- **US-006b**（簽名工具）← 需 US-003 + US-005b
- **US-007**（匯出）← 需 US-002 + US-005a + US-005b

### Wave 4（收尾）
- **US-009**（範本製作）← 需 US-004
