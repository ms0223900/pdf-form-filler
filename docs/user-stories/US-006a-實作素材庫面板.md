### US-006a：實作素材庫套用面板

**作為** 使用者
**我想要** 在填寫頁中開啟側邊欄選擇已儲存的個人資料，自動填入對應欄位
**以便** 減少重複輸入

**輸入格式**：
- IndexedDB 中的 `materials` 資料（個人資料）
- PDF 欄位名稱列表

**輸出格式**：
- `src/components/MaterialPanel.tsx` — 素材庫側邊欄（Sheet）
- `src/components/MaterialCard.tsx` — 單一素材卡片
- 更新 `src/app/fill/[id]/page.tsx` — 整合素材面板

**驗收條件**：
- [ ] 點擊按鈕可開啟素材庫側邊欄（Sheet 元件）
- [ ] 側邊欄顯示已儲存的個人資料素材列表
- [ ] 每筆素材顯示名稱與內容摘要
- [ ] 無素材時顯示提示引導前往素材管理頁新增
- [ ] 點擊個人資料素材 → 自動比對欄位名稱關鍵字並填入對應欄位
- [ ] 填入後欄位值即時反映在 UI 上
- [ ] `npx tsc --noEmit` 無錯誤

**依賴關係**：
- US-003（需 materialStore）
- US-005b（需欄位面板作為填入目標）

**優先級**：P1
**相關功能**：素材庫（spec §3.2 素材庫面板, §6）
