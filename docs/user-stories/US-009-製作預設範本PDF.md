### US-009：製作預設範本 PDF

**作為** 開發者
**我想要** 建立 2-3 個內建測試用範本 PDF
**以便** 使用者首次使用時有範例表單可選

**輸入格式**：
- 2-3 個含 AcroForm 欄位的測試用 PDF 檔案（使用工具製作，非程式碼產生）
- 已完成的 US-004 儀表板（PresetTemplates 已預留串接點）

**輸出格式**：
- `public/templates/` — 內建範本 PDF（含各式欄位型別的測試用表單）

**驗收條件**：
- [ ] `public/templates/` 包含至少 2 個 PDF 範本
- [ ] 至少一個範本同時包含文字欄位、核取方塊、下拉選單
- [ ] 首次造訪儀表板時可看到預設範本
- [ ] 點擊範本可載入並導航至填寫頁
- [ ] `npm run build` 無錯誤

**依賴關係**：
- US-004（需儀表板 PresetTemplates 已實作）

**優先級**：P2
**相關功能**：預設範本（spec §3.4, §8 step 10）
