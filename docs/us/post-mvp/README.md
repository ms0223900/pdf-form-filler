# Post-MVP — 加分功能

## 範圍說明

此階段接在 MVP 之後，補上**素材庫、簽名工具、素材管理頁、預設範本**，讓產品從「堪用」變成「好用」。

所有 US 皆依賴 MVP 已完成的基礎設施（US-001、US-002、US-004、US-005a、US-005b）。

## US 列表

| 編號 | 標題 | 優先級 | 依賴 | 說明 |
|------|------|--------|------|------|
| US-003 | 實作素材 CRUD 儲存層 | P0 | US-001 | materialStore CRUD |
| US-006a | 實作素材庫套用面板 | P1 | US-003, US-005b | 素材側邊欄與自動填入 |
| US-006b | 實作簽名繪製與儲存工具 | P1 | US-003, US-005b | Canvas 簽名繪製 |
| US-008 | 實作素材管理頁面 | P1 | US-003 | 素材 CRUD 頁面 |
| US-009 | 製作預設範本 PDF | P2 | US-004 | 內建範例 PDF |

## 依賴關係

```
 MVP 層（已完成）
 ┌──────────────────────────────────┐
 │ US-001, US-002, US-004, US-005a/b │
 └──────────────────────────────────┘
              │
     ┌────────┼────────┐
     ▼        ▼        ▼
  US-003   US-005b   US-004
     │        │        │
     ├──▶ US-006a ◀────┘
     ├──▶ US-006b
     └──▶ US-008
                       US-009 ◀──────┘
```

## 開發順序

```
Wave 1: US-003（materialStore，其他 US 的基礎）
Wave 2: US-008（素材管理頁）+ US-006a（素材套用）← 可平行
Wave 3: US-006b（簽名工具）
Wave 4: US-009（預設範本）
```

## 邊界（禁止事項）

Post-MVP 階段**不應**修改或重構 MVP 已完成的程式碼，除非是為了修正 MVP 期間未發現的 bug。所有新增功能應以**增量方式**疊加在 MVP 之上。
