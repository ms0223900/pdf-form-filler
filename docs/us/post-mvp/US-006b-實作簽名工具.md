### US-006b：實作簽名繪製與儲存工具

**作為** 使用者
**我想要** 在填寫頁中繪製簽名，並能儲存至素材庫供日後使用
**以便** 在 PDF 中加入簽名

**輸入格式**：
- 使用者在 canvas 上繪製的簽名筆跡
- IndexedDB 中的既有簽名素材

**輸出格式**：
- `src/components/SignaturePad.tsx` — 簽名繪製元件
- `src/components/SignaturePadDialog.tsx` — 簽名繪製對話框
- 更新 `src/app/fill/[id]/page.tsx` — 整合簽名功能

**驗收條件**：
- [x] 可開啟簽名繪製對話框
- [x] 在 canvas 上可用滑鼠/觸控繪製簽名
- [x] 支援調整畫筆粗細
- [x] 可清除簽名重畫
- [x] 繪製完成後簽名顯示在填寫頁中
- [x] 新簽名可儲存至素材庫
- [x] 可從素材庫選取既有簽名使用
- [x] 簽名資料以 base64 PNG 格式傳遞
- [x] `npx tsc --noEmit` 無錯誤

**依賴關係**：
- US-003（需 materialStore 儲存簽名）
- US-005b（需欄位面板作為簽名嵌入目標）

**優先級**：P1
**相關功能**：簽名工具（spec §3.2 簽名工具, §7.4）
