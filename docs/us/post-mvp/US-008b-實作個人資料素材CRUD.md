### US-008b：個人資料素材 CRUD

**作為** 使用者
**我想要** 在 `/materials` 頁面的「個人資料」分頁下，新增、編輯、刪除個人資料素材
**以便** 建立可重複使用的個人資料（姓名、電話、Email、地址、統編）

**輸入格式**：
- 使用者輸入的個人資料（素材名稱、姓名、電話、Email、地址、統編）
- IndexedDB 中的 `materials` 資料（透過 materialStore CRUD）
- US-008a 完成的頁面框架（4 tab 頁面、型別定義）

**輸出格式**：
- `src/components/PersonalInfoForm.tsx` — 個人資料編輯表單（支援新增與編輯模式）
- 更新 `src/app/materials/page.tsx` — 「個人資料」tab 顯示素材列表 + 新增按鈕

**驗收條件**：
- [ ] 「個人資料」tab 顯示已儲存素材的卡片列表（名稱、內容摘要如 姓名·電話·Email、建立時間）
- [ ] 點擊「新增」按鈕開啟 PersonalInfoForm
- [ ] PersonalInfoForm 表單欄位：素材名稱、姓名、電話、Email、地址、統編
- [ ] 表單驗證：素材名稱必填
- [ ] 儲存後列表即時更新
- [ ] 點擊既有素材可進入編輯模式（表單預填目前值）
- [ ] 編輯後儲存可更新素材
- [ ] 刪除需確認對話框
- [ ] 無個人資料素材時顯示提示引導新增
- [ ] `npx tsc --noEmit` 無錯誤

**依賴關係**：
- US-008a（需頁面框架與型別定義已就緒）

**優先級**：P1
**相關功能**：素材管理 — 個人資料（spec §3.3）
