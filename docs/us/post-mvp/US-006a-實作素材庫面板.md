### US-006a：實作素材庫套用面板

**作為** 使用者
**我想要** 在填寫頁中開啟側邊欄選擇已儲存的個人資料，自動填入對應欄位
**以便** 減少重複輸入

**輸入格式**：
- IndexedDB 中的 `materials` 資料（個人資料）
- PDF 欄位名稱列表

**輸出格式**：
- `src/components/MaterialPanel.tsx` — 素材庫側邊欄（Sheet），支援 `onNavigateToManage` callback 供空白狀態觸發彈窗
- `src/components/MaterialCard.tsx` — 單一素材卡片
- 更新 `src/app/fill/[id]/page.tsx` — 整合素材面板與素材管理彈窗

**驗收條件**：
- [x] 點擊按鈕可開啟素材庫側邊欄（Sheet 元件）
- [x] 側邊欄顯示已儲存的個人資料素材列表（分類顯示個人資料/簽名）
- [x] 每筆素材顯示名稱與內容摘要
- [x] 無素材時顯示提示引導，可前往素材管理頁或開啟滿版彈窗新增（透過 `onNavigateToManage` callback）
- [x] 點擊個人資料素材 → 在 PDF 目前頁面建立多個文字區塊（含欄位標籤與值）
- [x] 點擊簽名素材 → 在 PDF 目前頁面置中新增圖片區塊
- [x] `npx tsc --noEmit` 無錯誤

**依賴關係**：
- US-003（需 materialStore）
- US-005a（需填寫頁面框架與 PDF 預覽）

**優先級**：P1
**相關功能**：素材庫（spec §3.2 素材庫面板, §6）
