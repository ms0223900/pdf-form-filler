### US-005b：實作表單欄位面板

**作為** 使用者
**我想要** 在 PDF 預覽頁的右側看到自動偵測的表單欄位並填寫
**以便** 完成 PDF 填表

**輸入格式**：
- `PDFField[]`（來自 pdfUtils 的 `detectFormFields`）
- 使用者對各欄位的輸入值

**輸出格式**：
- `src/components/FormFieldPanel.tsx` — 欄位列表面板
- `src/components/FieldRenderer.tsx` — 依據欄位型別渲染對應 shadcn 元件
- 更新 `src/app/fill/[id]/page.tsx` — 整合欄位面板到預留位置

**驗收條件**：
- [x] 右側欄位面板顯示所有偵測到的 AcroForm 欄位
- [x] 欄位名稱作為標籤顯示
- [x] 文字欄位渲染為 `<Input>`
- [x] 核取方塊渲染為 `<Checkbox>`
- [x] 選項按鈕渲染為 `<RadioGroup>`
- [x] 下拉選單渲染為 `<Select>`
- [x] 唯讀欄位不產生輸入 UI（顯示標籤 + 唯讀標記）
- [x] PDF 無 AcroForm 欄位時顯示提示文字而非空白面板
- [x] 修改欄位值時即時反映在 React state 中
- [x] `npx tsc --noEmit` 無錯誤

**依賴關係**：
- US-002（需 pdfUtils 偵測欄位）
- US-005a（需填寫頁框架與 PDF 預覽）

**優先級**：P0
**相關功能**：PDF 填寫（spec §3.2, §5）
