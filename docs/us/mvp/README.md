# MVP — 最小可行產品

## 範圍說明

MVP 涵蓋完整的 PDF 填表閉環：**上傳 → 預覽 → 填寫 → 匯出下載**。不包含素材庫、簽名、素材管理等加分功能。

## US 列表

| 編號 | 標題 | 優先級 | 依賴 | 說明 |
|------|------|--------|------|------|
| US-001 | 資料庫與型別定義 | P0 | - | 共用型別 + Dexie schema |
| US-002 | 封裝 PDF 操作工具函式 | P0 | US-001 | pdf-lib 封裝 |
| US-004 | 實作儀表板與 PDF 上傳 | P0 | US-001 | 首頁、上傳、列表 |
| US-005a | 實作 PDF 預覽元件 | P0 | US-001 | PDFViewer + 填寫頁框架 |
| US-005b | 實作表單欄位面板 | P0 | US-002, US-005a | 欄位偵測與輸入渲染 |
| US-007 | 實作 PDF 匯出功能 | P1 | US-002, US-005a, US-005b | 匯出下載按鈕 |

## 依賴關係

```
US-001 (型別+DB)
  ├──▶ US-002 (pdfUtils)
  │     └──▶ US-005b (欄位面板) ──▶ US-007 (匯出)
  ├──▶ US-004 (儀表板)
  └──▶ US-005a (PDF預覽)
        └──▶ US-005b
```

## 開發順序

```
Wave 1: US-001
Wave 2: US-002 + US-004（平行）
Wave 3: US-005a
Wave 4: US-005b
Wave 5: US-007
```

## 邊界（禁止事項）

MVP 階段**不應**實作以下功能：
- ❌ 素材庫（materialStore）
- ❌ 素材套用面板
- ❌ 簽名繪製工具
- ❌ 素材管理頁面
- ❌ 預設範本
- ❌ 任何超出上傳→填寫→匯出流程的功能
